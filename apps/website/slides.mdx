# reslide

React + MDX ベースのプレゼンテーションフレームワーク

←→ キーでスライド移動 / Space で次へ / Escape で概要モード

---

## 特徴

::click

- **React ネイティブ** — Vue 依存なし、React 19 対応

::click

- **MDX** でスライドを記述、React コンポーネント埋め込み可能

::click

- **Vite ベース** — 高速 HMR の開発体験

::click

- **トランジション** — fade / slide-left / slide-right / slide-up / slide-down

::click

- **フレームワーク非依存** — Next.js / Vite / Remix 等で利用可能

## layout: two-cols

## 2カラムレイアウト

`layout="two-cols"` + `SlotRight` で左右にコンテンツを配置

```tsx
<Slide layout="two-cols">
  <div>左カラム</div>
  <SlotRight>
    <div>右カラム</div>
  </SlotRight>
</Slide>
```

<SlotRight>

### 組み込みレイアウト

- `default` — 標準
- `center` — 中央寄せ
- `two-cols` — 2カラム
- `image-right` / `image-left`
- `section` — セクション区切り
- `quote` — 引用

## キーボードショートカット

| キー       | 機能                             |
| ---------- | -------------------------------- |
| **← →**    | スライド移動                     |
| **Space**  | 次のクリック / スライド          |
| **Escape** | 概要モード（サムネイルグリッド） |
| **f**      | フルスクリーン                   |
| **p**      | プレゼンターウィンドウ           |
| **d**      | 描画モード（フリーハンド注釈）   |
| **c**      | 描画クリア（描画モード中）       |

---

## クリックナビゲーション

docswell 風の画面端クリックでスライド送り / 戻し

::click

画面の **左端15%** をクリック → 前のスライドへ

画面の **右端15%** をクリック → 次のスライドへ

中央70%はナビゲーションなし（コンテンツ操作を妨げない）

::click

← 今すぐ画面の左右端をホバーしてみてください！矢印が表示されます →

## その他の機能

::click

- **GlobalLayer** — 全スライド共通のヘッダー / フッター / ロゴ

::click

- **PresenterView** — 別ウィンドウでノート + タイマー表示（`p` キー）

::click

- **PDF エクスポート** — `Ctrl+P` で印刷 / PDF 保存

::click

- **テーマ** — CSS 変数ベース、default / dark テーマ同梱
