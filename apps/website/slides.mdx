# reslide

React + MDX ベースのプレゼンテーションフレームワーク

---

## 特徴

- **React ネイティブ** — Vue 依存なし
- **MDX** でスライドを記述
- **Vite ベース** — 高速な HMR

::click

- カスタム React コンポーネントを埋め込み可能
- CSS 変数でテーマカスタマイズ
