# reslide

React + MDX ベースのプレゼンテーションフレームワーク

← → キーでスライド移動 / Space で次へ / Escape で概要モード

---

## 特徴

::click

- **MDX** でスライドを記述 — Markdown の手軽さ + React コンポーネントの表現力

::click

- **フレームワーク非依存** — Next.js / Vite / Remix 等あらゆる React プロジェクトで動作

::click

- **Vite ベース** — 高速 HMR、トランジション、テーマ、PDF エクスポートを同梱

## 数式サポート — KaTeX

インライン数式: $E = mc^2$ や $a^2 + b^2 = c^2$

ディスプレイ数式: $$1 + 2 + \cdots + n = n(n+1)/2$$

`$...$` でインライン、`$$...$$` でディスプレイ数式を記述

## layout: section

# プレゼンテーション操作

キーボード・マウス・プレゼンターモードで快適な発表体験

---

## 操作方法

| キー       | 機能                             |
| ---------- | -------------------------------- |
| **← →**    | スライド移動                     |
| **Space**  | 次のクリック / スライド          |
| **Escape** | 概要モード（サムネイルグリッド） |
| **f**      | フルスクリーン                   |
| **p**      | プレゼンターウィンドウ           |
| **d**      | 描画モード（フリーハンド注釈）   |

画面の左端 / 右端クリックでもスライド送り・戻しが可能

---

## ツールと機能

::click

- **プレゼンターモード** — 別ウィンドウでノート + タイマー表示（`p` キー）

::click

- **描画モード** — フリーハンド注釈を重ねて表示（`d` キー）

::click

- **GlobalLayer** — 全スライド共通のヘッダー / フッター / ロゴ

::click

- **テーマ & PDF** — CSS 変数ベースのテーマ + `Ctrl+P` で PDF エクスポート
